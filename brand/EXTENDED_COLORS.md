## COLOR USAGE RULES

### By Context

| Context              | Palette to use                              |
| -------------------- | ------------------------------------------- |
| CPI donor reports    | CPI Primary + CPI Secondary                 |
| YPSA documents       | YPSA Navy + White only                      |
| ZarishHealth app     | ZarishHealth tokens                         |
| Joint CPI/YPSA docs  | CPI primary palette (neutral gray dominant) |
| NCD dashboard        | Program Colors + ZH neutrals                |
| Public health briefs | CPI brand + data vis palette 1              |
| Community materials  | CPI Red + CPI Purple (high contrast)        |
| Print collateral     | CMYK equivalents of all hex values          |

### Combining Colors (Safety Rules)

* Never pair CPI Red + CPI Purple as dominant colors (clashes)
* Never pair Magenta + CPI Red together
* Always ensure 4.5:1 contrast ratio minimum for text
* ZarishHealth teal never appears in CPI-branded documents
* Program colors always appear with neutral backgrounds (never stacked)
