# YPSA Logo Usage Guide

**Young Power in Social Action — Logo Assets**
*Version 1.1 | Updated*

---

## Logo Assets

### Available Files

| File | Description | Use Case |
| ---- | ----------- | -------- |
| `YPSA-LOGO.ai` | Original vector (Adobe Illustrator) | Print, editing |
| `YPSA-LOGO-PNG.png` | Raster, transparent | Web, digital |
| `YPSA-LOGO-JPG.jpg` | Raster, white bg | Documents |
| `YPSA_logo_white.svg` | White version | Dark backgrounds |
| `YPSA_logo_icon.svg` | Icon version | Favicon, small |

---

## Clear Space

Maintain clear space equal to **10% of logo width** on all sides.

---

## Aspect Ratio

**0.80 : 1** (width : height) — always maintain

---

## Minimum Sizes

| Context | Minimum Height |
|---------|--------------|
| Print | 10mm |
| Web | 40px |
| Favicon | 16px |

---

## Logo DON'Ts

* Never leave out either element (image or text)
* Never stretch or distort the logo
* Never alter colors
* Never use another typeface
* Never display at an angle (always horizontal)
* Never use on low-contrast backgrounds

---

## Bilingual Presentation

YPSA logo should always be presented with **both Bengali and English**:

```
[Icon]  ইপসা (top arc)
        YPSA (bottom arc)
```

For field materials, Bengali text should be **equal or larger** than English.

---

## Co-branding with CPI

When YPSA appears alongside CPI:

| Element | Rule |
|---------|------|
| Logo order | CPI left, YPSA right |
| Logo sizing | Equal height (baseline aligned) |
| Divider | Thin vertical line #D0C4C5 |
| Background | White for logo bar |
| Spacing | Minimum 24px between logos |

## Background Guidelines

### Approved Combinations

| Background | Logo Version |
|-----------|--------------|
| White | Navy on white |
| Light gray (#e8e8f5) | Navy on light |
| Navy (#0B0B45) | White on navy |
| Dark teal (#025259) | White on teal |
| Gunmetal (#00272C) | White on gunmetal |

### Never Use On

- Photographs without overlay
- Low contrast colors
- Patterns that compete with logo
