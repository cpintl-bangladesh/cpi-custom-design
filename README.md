# CPI Custom Design SKILL

## Complete Brand & Design System Package

### Community Partners International Bangladesh × YPSA × ZarishHealth

---

## What Is This?

This is a **SKILL package** — a structured folder of Markdown files that any AI assistant
can read to understand and apply the visual design standards for:

* **CPI Bangladesh** (Community Partners International)
* **YPSA** (Young Power in Social Action)
* **ZarishHealth / ZarishSphere EMR** (Digital health platform)

Upload the entire folder (or specific files) to an AI assistant, and it will have
everything it needs to create on-brand documents, presentations, web interfaces,
posters, dashboards, and clinical forms.

---

## How to Use This

### With Claude (claude.ai)

1. Zip this folder: `CPI-custom-design-SKILL.zip`
2. Upload the zip in a Claude conversation
3. Say: *"Please read the SKILL.md file and apply CPI brand guidelines to..."*

### With Any Other AI

1. Upload the relevant `.md` files
2. Reference specific files as needed (e.g., "use `brand/CPI_BRAND.md` for colors")
