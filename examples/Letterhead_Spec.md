# CPI Bangladesh Letterhead Specification

**Ready-to-use layout spec for official correspondence**

---

## CPI Bangladesh Letterhead

 **Page Size** : A4 (210 × 297mm)
 **Margins** : Top 40mm (header area), Left 25mm, Right 20mm, Bottom 20mm

### Header (Top 35mm)

* Left: CPI Primary Logo (with tagline), height 20mm, positioned 10mm from left
* Right: "COMMUNITY PARTNERS INTERNATIONAL" Raleway SemiBold 8pt, CPI Warm Gray
* Right below: Program name (if applicable), Raleway Regular 8pt
* Watermark: CPI leaf watermark, right side, 46% opacity grey

### Red Rule

* 1.5pt line, CPI Red (#D91E4D), full page width, at 35mm from top

### Body Area (35mm to 270mm)

* Font: Raleway Regular 11pt
* Color: CPI Black (#2D2926)
* Line spacing: 1.4
* Paragraphs: 6pt space after

### Footer (270mm to 297mm)

* Thin CPI Red rule (1pt) at 270mm
* Left: "Community Partners International Bangladesh"
* Left below: "Rabbani Building (4th Floor), Opposite S. K. Tower, Kurushkul Bypass Road, Cox's Bazar"
* Right: "+880 15 3420 6479 | info@cpintl.org | www.cpintl.org"
* Font: Raleway Regular 7pt, CPI Warm Gray

# Dashboard Report Specification

**For ZarishHealth program data exports / M&E summaries**

## NCD Program Dashboard (PDF/HTML)

### Layout: A4 Landscape (297 × 210mm)

**Section 1 — Header Bar (25mm)**

* Background: ZH Gun Metal (#00272C)
* Left: "zarish **health** " wordmark (white)
* Center: Report title (white, Inter SemiBold 14pt)
* Right: Period + Facility (white, Inter Regular 11pt)
* Right: CPI + YPSA logos small (white versions)

**Section 2 — KPI Row (40mm)**

* 4 equal columns, white cards with teal top border
* Each: Large number (JetBrains Mono 36pt, ZH Gun Metal) + label (Inter 11pt, gray)
* Metrics: Total Patients | Active on Treatment | Follow-up Due | New Registrations

**Section 3 — Charts (90mm)**

* Left 60%: Line chart — monthly enrollment trend by program
* Right 40%: Donut chart — patients by program distribution
* Colors: Per program palette (HCV=Mimosa, CVD=Salmon, etc.)

**Section 4 — Patient Table (45mm)**

* Compact table, Inter 9pt
* Columns: ID | Name | Program | Camp | Status | Last Visit | Next Visit
* Alternating rows: white / #F7F9EC
* Status badges: colored per status system
* Left border on each row: 3px program color

**Footer**

* ZH Teal rule 1pt
* "Generated by ZarishHealth EMR | CPI Bangladesh + YPSA | zarishsphere.com"
* Inter Regular 8pt, #4A5568
