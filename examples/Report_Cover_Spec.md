# Report Cover Specification

**Ready-to-use layout specs for reports and publications**
*CPI Bangladesh Programs*

---

## Standard Report Cover (A4 Portrait)

```
┌──────────────────────────────────────────────────┐
│ [Full-bleed photo — top 180mm]                │
│ [CPI Red overlay 40% or purple overlay]        │
│                                            │
│ [CPI Logo position - 15mm from top left]      │
├──────────────────────────────────────────────────┤
│ [White area — 117mm]                         │
│                                            │
│  PROGRAM REPORT TITLE          │ Raleway Bold 28pt   │
│  Subtitle/Period        │ Raleway Light 16pt │
│                       │ #948794 (gray)     │
│                       │                   │
│  [CPI Red rule — 2pt horizontal]            │
│                                            │
│  Cox's Bazar, Bangladesh  │ Raleway 11pt     │
│  Q1 2026              │ #948794          │
│                                            │
│  [CPI Logo — bottom left, 45mm]            │
│  [YPSA Logo if co-branded — right aligned]    │
└──────────────────────────────────────────────────┘
```

### Cover Specifications

| Element          | Value                      |
| ---------------- | -------------------------- |
| Page size       | A4 (210 × 297mm)          |
| Photo zone     | 180mm height              |
| Body zone    | 117mm height              |
| Margins      | 15mm all sides            |
| Photo bleed  | Full bleed to edges        |

## Donor Report Cover

```
┌─────────────────────────────────────┐
│  [Full-bleed impact photo]           │
│  [No color overlay - clean]           │
├─────────────────────────────────────┤
│                                   │
│  ANNUAL/PROGRESS REPORT            │  ←raleway Bold 28pt, CPI Purple
│  [Title]                         │
│  [Period]                        │  ←Raleway Light 14pt, CPI Warm Gray
│                                   │
│  ────���────────────────────────     │
│                                   │
│  [Logo array - CPI + YPSA + funder] │  ← White logos on muted background
│  If funded: include funder logo     │
│                                   │
│  [Date] [Location]               │  ← Arial 9pt, CPI Warm Gray
└─────────────────────────────────────┘
```

### Donor Cover Rules

* Always include funder logos as specified in grant agreement
* Use white logo versions on dark/navy strip if background is dark
* Maintain equal visual weight for all partner logos
* Never use funder logo as dominant element
