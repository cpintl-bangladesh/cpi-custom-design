## Chart / Data Visualization Style

### Colors for Charts

```
Primary series:   #D91E4D  (CPI Red)
Secondary:        #615E9B  (CPI Violet)
Tertiary:         #4298B5  (CPI Teal)
Quaternary:       #41273B  (CPI Purple)
Background fill:  #F5F3F4  (light warm)
Grid lines:       #D0C4C5  (CPI Light Gray)
Axis text:        #948794  (CPI Warm Gray)
```

### Chart Rules

* Bar charts: Use CPI Red as primary fill
* Line charts: CPI Violet primary, CPI Red secondary
* Pie/donut: Use full CPI + secondary palette in order
* No gradients on chart fills — flat colors only
* Always include data labels in Raleway Regular 9pt
* Chart titles in Raleway SemiBold 12pt, CPI Purple
